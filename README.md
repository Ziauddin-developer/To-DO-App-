# Flutter Web SQLite Demo  A Flutter Web project demonstrating how to use SQLite in the browser for local data storage. This project is designed to showcase offline-first database functionality using modern web-compatible SQLite solutions.  ## ✨ Features  * Cross-platform Flutter application * Web-compatible SQLite integration * Local data persistence * Offline-first architecture * Clean and maintainable codebase * Responsive UI  ## 🛠️ Tech Stack  * Flutter * Dart * SQLite (Web Compatible) * Drift / SQLite WASM (if applicable)  ## 🎯 Purpose  The purpose of this project is to demonstrate how SQLite can be used in Flutter Web applications for local storage without requiring a backend server. It serves as a learning resource for developers exploring offline data management in Flutter.  ## 🚀 Getting Started  ```bash flutter pub get flutter run -d chrome ```  ## 📂 Use Cases  * Offline web applications * Personal productivity tools * Note-taking apps * Inventory management * Learning Flutter Web database integration  ## 📄 License  This project is open-source and available under the MIT License.
